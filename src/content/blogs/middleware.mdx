- Content delivery network service from AWS
- Enables you to serve content directly to AWS edge locations, nearest to the user's location.
- Source can be from _S3, EC2, ELB and External Websites_.
- :LiCross: If content is not available at edge location, it is retrieved from the origin server and cached.
- No minimum usage commitment
    - Users are not required to commit to a certain level of usage or spend a minimum amount each month. Instead, _you only pay for what you use_.
- Provides features to protect your private content.
